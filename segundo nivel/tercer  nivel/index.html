¿Qué necesitas para empezar a montar el entorno?

Antes de empezar, asegúrate de tener instalado en tu equipo las siguientes herramientas:

JDK7 o superior. Un cliente GIT: Si vas a utilizar Eclipse como IDE, te recomiendo usar el plugin egit. En este link puedes encontrar información de cómo descargarlo e instalarlo . Maven 3.x: Puedes descargar una distribución en este link. Te dejo algunos enlaces que te pueden resultar de ayuda a la hora de instalar Maven:

Ayuda oficial de instalación Instalación de Maven en Mac OS X Instalación del plugin de Maven para eclipse

Tomcat 8 (puedes utilizar otro si prefieres), como servidor para desplegar y probar la aplicación java Una cuenta en Bitbucket para acceder al repositorio git donde se encuentra la aplicación de ejemplo
